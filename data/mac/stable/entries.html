OSX Support Update

The 2.0.0 update for the I2P OSX Easy-Install Bundle will be delayed, because our OSX maintainer zlatinb recently resigned his position. We are working to replace the expertise, hardware, and credentials required to build and release updates.

Our current estimate is that the update will be available in about a month. We will make an annoucement here in the news feed when it is ready. You may check our development forum for the latest status. Thanks for your patience.

I2P Mac Bundle with Java 18.0.1 Security Update

A security vulnerability affecting the Java Runtime Environment used by I2P has been disclosed. Please update your Mac bundle to I2P 1.7.1 as soon as you can.

I2P Mac Bundle with Java 17.0.1

Version 1.5.1 of the I2P Mac DMG bundle is now available. It features the latest JDK 17.0.1. Please update.
